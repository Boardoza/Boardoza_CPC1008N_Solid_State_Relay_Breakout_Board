# Boardoza CPC1008N Solid State Relay Breakout Board

The **Boardoza CPC1008N Solid State Relay Breakout Board** is designed to provide safe, silent, and reliable switching for low-power **AC and DC loads** in maker and embedded projects. Based on the **CPC1008N Solid State Relay (SSR)**, it supports up to **100 Vp blocking voltage**, **150 mA continuous load current**, and offers **1500 Vrms optical isolation** between the input and output, ensuring dependable performance and improved system safety.

With its compact and efficient design, this breakout board is well suited for applications such as **multiplexers, data acquisition systems, electronic switching, I/O subsystems, and utility meters (electricity, water, and gas)**. It is also widely used in **medical devices, security systems, and industrial automation projects**, making it a modern, maintenance-free alternative to traditional reed relays for both hobbyists and professional developers.

## Key Features

- **Solid State Relay (SSR):** Provides safe, silent, and reliable switching for low-power AC and DC loads.
- **AC/DC Output Support:** Suitable for switching both alternating and direct current loads.
- **Optical Isolation:** Ensures electrical isolation between input and output for improved safety.
- **No Mechanical Wear:** Fully electronic switching with no contact bounce or lifetime limitations.
- **Low EMI/RFI:** Generates no electromagnetic or radio frequency interference during operation.

## Technical Specifications

**Model:** CPC1008N  
**Manufacturer:** Boardoza  
**Manufacturer IC:** IXYS / Littelfuse  
**Functions:** Solid State Relay (AC/DC Output)  
**Input Control Current:** 50 mA – 1 A  
**Blocking Voltage:** 100 Vp  
**Continuous Load Current:** 150 mA  
**Isolation Voltage:** 1500 Vrms  
**Off-State Leakage Current:** 1 µA @ 100 V  
**Operating Temperature:** -40 °C to +85 °C  
**Board Dimensions:** 20 mm × 20 mm
